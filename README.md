# Billows

**常用的工具函数集合**

## Install

```bash
npm install billows
```

## Usage

```ts
// 全量引入
import * as billows from 'billows';
billows.typeOf('chengbotao'); // "string"

// 差量使用
import { typeOf } from 'billows';
typeOf('chengbotao'); // "string"
```
